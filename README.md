## 下载

- 使用 IDE 内置插件系统：

  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>搜索 "Zhihu-Reader"</kbd> >
  <kbd>Install</kbd>
